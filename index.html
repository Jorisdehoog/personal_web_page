HELLO AND WELCOME

This is my site

This is my personal webpage, and is used as a playground for me to learn and get up to speed with simple web development. The simple reasoning for building this website from scratch, without using templates or site generators (such as Jekyll) is that I want to know how this works.

The main goals of this site are the following:

Programming Hosting Linux API
Developing an attractive website using HTML5, Javascript and CSS. This website is built from scratch using these tools. The projects tab serves as my playground for interactive web-development, where show my efforts in graphing, data analysis and user input methods.
This website is locally hosted on a first generation Raspberry pi, running Debian Stretch Lite. Security of the server is a primary goal of this project. Secure transfer of the site to the browser is also taken care of.
As the server is running a CLI Debian Stretch distro, it is an excellent platform for me to learn how to handle a mininal Linux OS. I'm also running Ubuntu 18.04 LTE on my main PC which serves as my bread-and-butter Linux environment for all my dev needs, but I haven't left the Windows environment entirely.
Finally, in the scope of building this website, another goal of mine is learning how to interface with other recources by using API's.
